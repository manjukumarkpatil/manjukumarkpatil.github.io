MS Research Project

Stem-Computing using Drones.

Teaching STEM concepts using interactive learning tools would benefit students for visualizing the concepts in more intuitive way. Traditional ways of teaching for linear algebra concepts such as linear equations, quadratic equations and their associated graphs are not sufficient to reach students deeply with these concepts. However, with the use of technology, we can make the curriculum fun, interactive and link the real-world applications of these concepts, and make students engage deeply into the curriculum. For this purpose, designed and developed a STEM educational curriculum demonstrating parabolic equations using drones through Face Detection algorithm. Implemented DJI drone drivers for middleware, and used JavaScript, PHP for the web interface.
